4 Information requirements

This clause specifies the information requirements for the Modified swept solid application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Modified_swept_solid_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Modified_swept_solid_arm;
(*

4.1 Required AM ARMs

The following EXPRESS interface statements specify the elements imported from the ARMs of other application modules.

EXPRESS specification:

*)
USE FROM Basic_geometric_topology_arm;    --  ISO/TS 10303-1323

USE FROM Swept_solid_arm;    --  ISO/TS 10303-1321
(*

NOTE 1   The schemas referenced above are specified in the following part of ISO 10303:

Basic_geometric_topology_arm ISO/TS 10303-1323
Swept_solid_arm ISO/TS 10303-1321

NOTE 2   See Annex C, Figures C.1and C.2 for a graphical representation of this schema.

4.2 ARM type definitions

This subclause specifies the ARM types for this application module. The ARM types and definitions are specified below.

4.2.1 trim_intent   

A trim_intent is an enumeration of the descriptive specification of the users requirements for end condtions when trimming an ble select type used to specify the type of end condition required when trimming an Extruded_solid.

EXPRESS specification:

*)
TYPE trim_intent = ENUMERATION OF
   (blind,
    through_all,
    up_to_next,
    unspecified);
END_TYPE;
(*

Enumerated item definitions:

blind: specifies that the extrusion terminates before it meets any other face;

through_all: specifies that the extrusion extends through the entire model;

up_to_next: specifies that the extrusion terminates when it first meets a face;

unspecified: no intent is specified.

4.2.2 trim_select   

A trim_select is a select type used to specify the type of end condition required when trimming an Extruded_solid.

EXPRESS specification:

*)
TYPE trim_select = SELECT
   (Advanced_face,
    Detailed_geometric_model_element,
    length_measure,
    plane_angle_measure);
END_TYPE;
(*

4.3 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.3.1 Extruded_solid_with_draft_angle   

An Extruded_solid_with_draft_angle is a type of Extruded_solid_with_trim_conditions to which a draft angle is applied along the length of the extrusion. The cross-section of the extrusion in the plane of the defining face is unaltered by the draft angle. At any point on the faces generated by the extrusion process the draft angle defines the angle between the tangent plane to the surface and the direction of extrusion. If the draft angle is positive the cross sectional area will increase in the positive direction of extrusion.

EXPRESS specification:

*)
ENTITY Extruded_solid_with_draft_angle
  SUBTYPE OF (Extruded_solid_with_trim_conditions);
  draft_angle : plane_angle_measure;
END_ENTITY;
(*

Attribute definitions:

draft_angle: the angle between the tangent plane to the surfaces generated by the extrusion process and the extruded_direction.

4.3.2 Extruded_solid_with_multiple_draft_angles   

An Extruded_solid_with_multiple_draft_angles is a type of Extruded_solid_with_trim_conditions to which draft angles are applied along the length of the extrusion. The assigned draft angles are not the same for all edges of the swept face.

EXPRESS specification:

*)
ENTITY Extruded_solid_with_multiple_draft_angles
  SUBTYPE OF (Extruded_solid_with_trim_conditions);
  draft_edges : LIST[2:?] OF SET[1:?] OF Edge_curve;
  draft_angles : LIST[2:?] OF plane_angle_measure;
WHERE
  WR1: SIZEOF(draft_edges) = SIZEOF(draft_angles);
  WR2: SIZEOF(QUERY(q <* draft_angles | q = 0)) = 0;
END_ENTITY;
(*

Attribute definitions:

draft_edges: a list of sets of edges of the extruded face, each edge in the set having the same draft angle. The nth. draft angle is associated with the nth set of draft edges.

draft_angles: the list of draft angles to be assigned to the sets of edges in the draft_edges set.

Formal propositions:

WR1: The number of draft_angles shall be equal to the number of sets of draft_edges.

WR2: The draft_angles shall all be non zero.

4.3.3 Extruded_solid_with_trim_conditions   

An Extruded_solid_with_trim_conditions is a type of Extruded_solid that has been modified by trimming to change the extruded length. An Extruded_solid_with_trim_conditions may be an Extruded_solid_with_draft_angle or an Extruded_solid_with_multiple_draft_angles.

EXPRESS specification:

*)
ENTITY Extruded_solid_with_trim_conditions
  SUPERTYPE OF (ONEOF (Extruded_solid_with_draft_angle,
                       Extruded_solid_with_multiple_draft_angles))
  SUBTYPE OF (Extruded_solid);
  start_condition : trim_select;
  end_condition : trim_select;
  start_condition_intent : trim_intent;
  end_condition_intent : trim_intent;
END_ENTITY;
(*

Attribute definitions:

start_condition: the required alternative starting point for the linear extrusion process to create the solid. This may be specified either as length_measure or via an intersecting face.

NOTE 1   a plane_angle_measure is not a valid trim condition for this entity.

NOTE 2   This entity is fully defined in the corresponding MIM object which contains some additional constraints.

end_condition: the required alternative termination point for the extrusion process to create the solid. This may be specified either as a length measure or via an intersecting face.

start_condition_intent: the intended semantics of the first trimming process.

end_condition_intent: the intended semantics of the final trimming process.

4.3.4 Solid_of_revolution_with_trim_conditions   

A Solid_of_revolution_with_trim_conditions is a type of Solid_of_revolution that has been modified by trimming to change the angle of revolution.

NOTE    This entity is fully defined in the corresponding MIM object which contains some additional constraints.

EXPRESS specification:

*)
ENTITY Solid_of_revolution_with_trim_conditions
  SUBTYPE OF (Solid_of_revolution);
  start_condition : trim_select;
  end_condition : trim_select;
END_ENTITY;
(*

Attribute definitions:

start_condition: the required alternative starting point for the revolution process to create the solid. This may be specified either as an angle or via an intersecting face.

NOTE    a length_measureis not a valid trim condition for this entity.

end_condition: the required alternative termination point for the revolution process to create the solid. This may be specified either as an angle or via an intersecting face.



*)
END_SCHEMA;  -- Modified_swept_solid_arm
