4 Information requirements

This clause specifies the information requirements for the Part template non planar shape application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Part_template_non_planar_shape_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Part_template_non_planar_shape_arm;
(*

EXPRESS specification:

*)

4.2 ARM type definition

This subclause specifies the ARM type for this application module. The ARM type and definition is specified below.

4.2.1 ptnps_part_template_or_physical_unit_3d_shape_model_select   

The ptnps_part_template_or_physical_unit_3d_shape_model_select type is an extension of the part_template_or_physical_unit_3d_shape_model_select type. It adds the data type Part_template_non_planar_2d_shape_model to the list of alternate data types.

NOTE  The list of entity data types may be extended in application modules that use the constructs of this module.

EXPRESS specification:

*)
TYPE ptnps_part_template_or_physical_unit_3d_shape_model_select = EXTENSIBLE GENERIC_ENTITY SELECT BASED_ON part_template_or_physical_unit_3d_shape_model_select WITH
   (Part_template_non_planar_2d_shape_model);
END_TYPE;
(*

4.3 ARM entity definition

This subclause specifies the ARM entity for this module. The ARM entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entity and definition is specified below.

4.3.1 Part_template_non_planar_2d_shape_model   

A Part_template_non_planar_2d_shape_model is a type of Manifold_surface_shape_representation and a type of Part_template_shape_model. A Part_template_non_planar_2d_shape_model is the shape of a Template_definition that is represented in the form of a surface.

EXPRESS specification:

*)
ENTITY Part_template_non_planar_2d_shape_model
  SUBTYPE OF (Manifold_surface_shape_representation, Part_template_shape_model);
  transformation : Axis_placement_3d;
WHERE
  WR1: NOT EXISTS(SELF\Representation.name);
  WR2: NOT EXISTS(SELF\Representation.description);
END_ENTITY;
(*

Attribute definitions:

transformation: specifies a role of the Axis_placement_3d for the Part_template_non_planar_2d_shape_model.

Formal propositions:

WR1: The name shall not be populated.

WR2: The description shall not be populated.



*)
END_SCHEMA;  -- Part_template_non_planar_shape_arm
