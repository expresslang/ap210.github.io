4 Information requirements

This clause specifies the information requirements for the Pre defined datum 2D symbol application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Pre_defined_datum_2d_symbol_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Pre_defined_datum_2d_symbol_arm;
(*

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Pre_defined_datum_2d_symbol   

A Pre_defined_datum_2d_symbol is a type of Planar_shape_model. A Pre_defined_datum_2d_symbol is either a Pre_defined_parallel_datum_axis_2d_symbol, or a Pre_defined_perpendicular_datum_axis_2d_symbol or a Pre_defined_perpendicular_datum_plane_2d_symbol.

EXPRESS specification:

*)
ENTITY Pre_defined_datum_2d_symbol
  ABSTRACT SUPERTYPE OF (ONEOF (Pre_defined_parallel_datum_axis_2d_symbol,
                                Pre_defined_perpendicular_datum_axis_2d_symbol,
                                Pre_defined_perpendicular_datum_plane_2d_symbol))
  SUBTYPE OF (Planar_shape_model);
  associated_element : Datum_axis;
WHERE
  WR1: NOT EXISTS(SELF\Representation.description);
END_ENTITY;
(*

Attribute definitions:

associated_element: specifies a role of the Datum_axis for the Pre_defined_datum_2d_symbol.

Formal propositions:

WR1: The description shall not be populated.

4.2.2 Pre_defined_parallel_datum_axis_2d_symbol   

A Pre_defined_parallel_datum_axis_2d_symbol is a type of Pre_defined_datum_2d_symbol. A Pre_defined_parallel_datum_axis_2d_symbol is a standard representation for a Datum_axis in a 2 dimensional coordinate system. The axis represented shall be parallel to, or congruent with, the plane that represents the 2 dimensional coordinate system in an associated 3 dimensional coordinate system.

EXPRESS specification:

*)
ENTITY Pre_defined_parallel_datum_axis_2d_symbol
  SUBTYPE OF (Pre_defined_datum_2d_symbol);
END_ENTITY;
(*

4.2.3 Pre_defined_perpendicular_datum_axis_2d_symbol   

A Pre_defined_perpendicular_datum_axis_2d_symbol is a type of Pre_defined_datum_2d_symbol. A Pre_defined_perpendicular_datum_axis_2d_symbol is a standard representation for a Datum_axis that is parallel to the normal of the plane that represents the 2 dimensional coordinated system in an associated 3 dimensional coordinate system.

EXPRESS specification:

*)
ENTITY Pre_defined_perpendicular_datum_axis_2d_symbol
  SUBTYPE OF (Pre_defined_datum_2d_symbol);
END_ENTITY;
(*

4.2.4 Pre_defined_perpendicular_datum_plane_2d_symbol   

A Pre_defined_perpendicular_datum_plane_2d_symbol is a type of Pre_defined_datum_2d_symbol. A Pre_defined_perpendicular_datum_plane_2d_symbol is a standard representation for a Datum_plane that is perpendicular to the plane that represents the 2 dimensional coordinate system in an associated 3 dimensional coordinate system.

EXPRESS specification:

*)
ENTITY Pre_defined_perpendicular_datum_plane_2d_symbol
  SUBTYPE OF (Pre_defined_datum_2d_symbol);
END_ENTITY;
