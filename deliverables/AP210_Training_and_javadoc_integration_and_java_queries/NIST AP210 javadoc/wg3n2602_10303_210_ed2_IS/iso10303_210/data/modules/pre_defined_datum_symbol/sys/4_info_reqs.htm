4 Information requirements

This clause specifies the information requirements for the Pre defined datum symbol application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Pre_defined_datum_symbol_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Pre_defined_datum_symbol_arm;
(*

4.1 Required AM ARMs

The following EXPRESS interface statements specify the elements imported from the ARMs of other application modules.

EXPRESS specification:

*)
USE FROM Layered_3d_shape_arm;    --  ISO/TS 10303-1694

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship   

A Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship is a type of Shape_definition_3d_intersection. A Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship establishes the geometric reference relationship between a Datum_axis representation in a 3 dimensional coordinate system and the related representation in a 2 dimensional coordinate system where the Datum_axis is parallel to the plane that represents the 2 dimensional coordinate system in the 3 dimensional coordinate system.

NOTE    Only the line representing the central axis is expected to be mathematically derived. Due to differences in 2 dimensional CAD systems the start and end treatments (arrowheads) may not be mathematically derived.

EXPRESS specification:

*)
ENTITY Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship
  SUBTYPE OF (Shape_definition_3d_intersection);
  SELF\Shape_definition_3d_intersection.surface_intersection_result : Pre_defined_parallel_datum_axis_2d_symbol;
  SELF\Shape_definition_3d_intersection.projected_volume : Pre_defined_datum_axis_3d_symbol;
END_ENTITY;
(*

Attribute definitions:

surface_intersection_result: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_parallel_datum_axis_2d_symbol for the Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship.

projected_volume: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_datum_axis_3d_symbol for the Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship.

4.2.2 Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship   

A Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship is a type of Shape_definition_3d_intersection. A Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship establishes the geometric reference relationship between a Datum_axis representation in a 3 dimensional coordinate system and the related representation in a 2 dimensional coordinate system where the Datum_axis is perpendicular to the plane that represents the 2 dimensional coordinate system in the 3 dimensional coordinate system. The controlled information is the centre of the circle.

NOTE    Only the center of the circle representing the central axis is expected to be directly referenced. Due to differences in CAD systems resolutions the lines used as crosshairs may sightly deviate from intersecting at the center of the circle.

EXPRESS specification:

*)
ENTITY Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship
  SUBTYPE OF (Shape_definition_3d_intersection);
  SELF\Shape_definition_3d_intersection.surface_intersection_result : Pre_defined_perpendicular_datum_axis_2d_symbol;
  SELF\Shape_definition_3d_intersection.projected_volume : Pre_defined_datum_axis_3d_symbol;
END_ENTITY;
(*

Attribute definitions:

surface_intersection_result: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_perpendicular_datum_axis_2d_symbol for the Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship.

projected_volume: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_datum_axis_3d_symbol for the Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship.

4.2.3 Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship   

A Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship is a type of Shape_definition_3d_intersection. A Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship establishes the geometric reference relationship between a Datum_plane representation in a 3 dimensional coordinate system and the related representation in a 2 dimensional coordinate system where the Datum_plane is perpendicular to the plane that represents the 2 dimensional coordinate system in the 3 dimensional coordinate system. The controlled information is the line representing the normal of the plane.

NOTE    Due to differences in CAD systems resolutions the lines used as base and arrowhead may sightly deviate from intersecting at the start and end of the line.

EXPRESS specification:

*)
ENTITY Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship
  SUBTYPE OF (Shape_definition_3d_intersection);
  SELF\Shape_definition_3d_intersection.surface_intersection_result : Pre_defined_perpendicular_datum_plane_2d_symbol;
  SELF\Shape_definition_3d_intersection.projected_volume : Pre_defined_datum_plane_3d_symbol;
END_ENTITY;
(*

Attribute definitions:

surface_intersection_result: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_perpendicular_datum_plane_2d_symbol for the Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship.

projected_volume: an attribute inherited from the Shape_definition_3d_intersection shall be redeclared as the Pre_defined_datum_plane_3d_symbol for the Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship.

4.3 ARM subtype constraint definitions

This subclause specifies the ARM subtype constraints for this module. Each subtype constraint places constraints on the possible super-type / subtype instantiations. The ARM subtype constraints and definitions are specified below.

4.3.1 pdds_non_feature_shape_model_subtypes   

The pdds_non_feature_shape_model_subtypes constraint specifies a constraint that applies to instances of subtypes of Non_feature_shape_model.

EXPRESS specification:

*)
SUBTYPE_CONSTRAINT pdds_non_feature_shape_model_subtypes FOR Non_feature_shape_model;
  ONEOF (Design_view_level_non_feature_shape_model,
         Usage_view_level_non_feature_shape_model);
END_SUBTYPE_CONSTRAINT;
(*

4.3.2 pdds_shape_definition_3d_intersection_subtypes   

The pdds_shape_definition_3d_intersection_subtypes constraint specifies a constraint that applies to instances of subtypes of Shape_definition_3d_intersection.

EXPRESS specification:

*)
SUBTYPE_CONSTRAINT pdds_shape_definition_3d_intersection_subtypes FOR Shape_definition_3d_intersection;
  ONEOF (Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship,
         Pre_defined_perpendicular_datum_axis_symbol_3d_2d_relationship,
         Pre_defined_parallel_datum_axis_symbol_3d_2d_relationship);
END_SUBTYPE_CONSTRAINT;
