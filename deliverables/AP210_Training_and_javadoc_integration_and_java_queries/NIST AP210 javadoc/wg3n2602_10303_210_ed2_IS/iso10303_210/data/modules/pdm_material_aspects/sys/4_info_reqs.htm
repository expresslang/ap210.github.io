4 Information requirements

This clause specifies the information requirements for the Pdm material aspects application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Pdm_material_aspects_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Pdm_material_aspects_arm;
(*

4.1 Required AM ARMs

The following EXPRESS interface statements specify the elements imported from the ARMs of other application modules.

EXPRESS specification:

*)
USE FROM Generic_material_aspects_arm;    --  ISO/TS 10303-1681

USE FROM File_identification_arm;    --  ISO/TS 10303-1127

USE FROM Part_view_definition_arm;    --  ISO/TS 10303-1023

USE FROM Product_as_individual_arm;    --  ISO/TS 10303-1164
(*

NOTE 1   The schemas referenced above are specified in the following part of ISO 10303:

Generic_material_aspects_arm ISO/TS 10303-1681
File_identification_arm ISO/TS 10303-1127
Part_view_definition_arm ISO/TS 10303-1023
Product_as_individual_arm ISO/TS 10303-1164

NOTE 2   See Annex C, Figures C.1and C.2 for a graphical representation of this schema.

4.2 ARM type definition

This subclause specifies the ARM type for this application module. The ARM type and definition is specified below.

4.2.1 pdmma_material_item_select   

The pdmma_material_item_select type is an extension of the material_item_select type. It adds the data types Hardcopy, Part_view_definition and Product_as_individual to the list of alternate data types.

NOTE  The list of entity data types may be extended in application modules that use the constructs of this module.

EXPRESS specification:

*)
TYPE pdmma_material_item_select = EXTENSIBLE GENERIC_ENTITY SELECT BASED_ON material_item_select WITH
   (Hardcopy,
    Part_view_definition,
    Product_as_individual);
END_TYPE;
(*



*)
