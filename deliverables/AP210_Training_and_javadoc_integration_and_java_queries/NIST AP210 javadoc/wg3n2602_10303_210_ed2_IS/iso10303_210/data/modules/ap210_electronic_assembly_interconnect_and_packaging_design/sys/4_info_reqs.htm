EXPRESS specification:

*)

(*

4.4 ARM rule definitions

This subclause specifies the ARM rules for this module. The ARM rules and definitions are specified below.

4.4.1 area_component_shape_constraint

The rule area_component_shape_constraint constrains the type of an Assembly_component_2d_shape_model when the Assembly_component is an Area_component.

EXPRESS specification:

*)
RULE area_component_shape_constraint FOR
(Area_component);
WHERE
  WR1: SIZEOF( QUERY( ac <* Area_component | NOT(SIZEOF (QUERY (ac2ds <* USEDIN(ac,'PHYSICAL_UNIT_2D_DESIGN_VIEW_ARM.ASSEMBLY_COMPONENT_2D_SHAPE_MODEL.SHAPE_CHARACTERIZED_COMPONENT') | NOT('CONSTRUCTIVE_SOLID_GEOMETRY_2D_ARM.SINGLE_AREA_CSG_2D_MODEL' IN TYPEOF(ac2ds)) )) = 0) )) = 0;
  WR2: SIZEOF( QUERY( ac <* Area_component | NOT(ac\Area_component.is_base) OR NOT(SIZEOF (QUERY (ac2ds <* USEDIN(ac,'PHYSICAL_UNIT_2D_DESIGN_VIEW_ARM.ASSEMBLY_COMPONENT_2D_SHAPE_MODEL.SHAPE_CHARACTERIZED_COMPONENT') | NOT('CONSTRUCTIVE_SOLID_GEOMETRY_2D_ARM.SINGLE_BOUNDARY_CSG_2D_MODEL' IN TYPEOF(ac2ds)) )) = 0) )) = 0;
END_RULE;
(*

Argument definitions:

Area_component : the set of all instances of Area_component.

Formal propositions:

WR1: If the Area_component is represented by an Assembly_component_2d_shape_model, the type of the shape shall be Single_area_csg_2d_model.

WR2: If the Area_component is represented by an Assembly_component_2d_shape_model and is a base component, the type of the shape shall be a Single_boundary_csg_2d_model.

4.4.2 physical_unit_network_definition_constraint

The rule physical_unit_network_definition_constraint specifies that each Physical_unit_network_definition that is referenced by a Physical_unit_shape_model shall be in a complex with an Assembly_module_design_view.

EXPRESS specification:

*)
RULE physical_unit_network_definition_constraint FOR
(Physical_unit_network_definition);
WHERE
  WR1: SIZEOF ( QUERY ( fund <* Physical_unit_network_definition | ('ASSEMBLY_MODULE_DESIGN_ARM.' + 'ASSEMBLY_MODULE_DESIGN_VIEW' IN TYPEOF (fund)) OR (SIZEOF(USEDIN(fund,'PHYSICAL_UNIT_SHAPE_WITH_PARAMETERS_ARM.' + 'PHYSICAL_UNIT_SHAPE_MODEL.SHAPE_CHARACTERIZED_DEFINITION')) = 0) )) = SIZEOF(Physical_unit_network_definition);
END_RULE;
(*

Argument definitions:

Physical_unit_network_definition : the set of all instances of Physical_unit_network_definition.

Formal propositions:

WR1: Each Physical_unit_network_definition that is referenced by a Physical_unit_shape_model shall be in a complex with an Assembly_module_design_view.



*)
