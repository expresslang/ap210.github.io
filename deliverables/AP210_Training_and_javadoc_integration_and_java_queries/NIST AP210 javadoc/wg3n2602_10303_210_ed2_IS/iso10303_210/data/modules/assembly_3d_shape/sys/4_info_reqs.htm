4 Information requirements

This clause specifies the information requirements for the Assembly 3D shape application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Assembly_3d_shape_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Assembly_3d_shape_arm;
(*

USE FROM Assembly_technology_arm;    --  ISO/TS 10303-1649

4.2 ARM entity definition

This subclause specifies the ARM entity for this module. The ARM entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entity and definition is specified below.

4.2.1 Bond_assembly_position   

A Bond_assembly_position is the placement of an Assembled_with_bonding within the Physical_unit_3d_shape_model of an assembly. The Bond_assembly_position is an approximation to the actual position since it uses a default shape for the bond. There may be considerable variation between the default shape and actual shapes in the "as delivered" product.

EXPRESS specification:

*)
ENTITY Bond_assembly_position;
  bond : Assembled_with_bonding;
  assembly_model : Physical_unit_3d_shape_model;
  bond_model : Default_assembly_bond_shape_model;
  bond_definition_placement : Axis_placement_3d;
UNIQUE
  UR1: bond;
END_ENTITY;
(*

Attribute definitions:

bond: specifies the role of the Assembled_with_bonding for the Bond_assembly_position.

assembly_model: specifies the Physical_unit_3d_shape_model of the assembly for the Bond_assembly_position.

bond_model: specifies the Default_assembly_bond_shape_model of the assembly bond for the Bond_assembly_position.

bond_definition_placement: specifies the role of the Axis_placement_3d for the Bond_assembly_position. The placement specifies the transformation applied to the bond model to bring it into the assembly model.

Formal propositions:

UR1: the bond shall be unique within a population of Bond_assembly_position.



*)
END_SCHEMA;  -- Assembly_3d_shape_arm
