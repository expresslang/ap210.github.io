TECHNICAL SPECIFICATION ISO/TS 10303-1659:2006(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1659:
Application module: Curve swept solid

1 Scope

This part of ISO 10303 specifies the application module Curve swept solid. The following are within the scope of this part of ISO 10303:

  • the definition of an extruded solid along a b-spline curve as centerline and a fixed cross-section plus orientation;
  • items within the scope of application module Elemental geometric shape, ISO/TS 10303-1004.

The following are outside the scope of this part of ISO 10303:

  • extruded solids with a centerline curve other than b-spline.
