TECHNICAL SPECIFICATION ISO/TS 10303-1231:2009(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1231:
Application module: Product data management

1 Scope

This part of ISO 10303 specifies the application module Product data management. The following are within the scope of this part of ISO 10303:

  • identification;
  • versioning;
  • structures;
  • transformations;
  • approvals;
  • work orders;
  • work requests;
  • effectivities;
  • classification; and
  • properties;
  • items within the scope of application module Assembly structure, ISO/TS 10303-1026;
  • items within the scope of application module Configuration effectivity, ISO/TS 10303-1058;
  • items within the scope of application module Document management, ISO/TS 10303-1290;
  • items within the scope of application module Effectivity application, ISO/TS 10303-1059;
  • items within the scope of application module External class, ISO/TS 10303-1275;
  • items within the scope of application module Incomplete data reference mechanism, ISO/TS 10303-1349;
  • items within the scope of application module Independent property representation, ISO/TS 10303-1038;
  • items within the scope of application module Process property assignment, ISO/TS 10303-1040;
  • items within the scope of application module Product structure, ISO/TS 10303-1134;
  • items within the scope of application module Product view definition properties, ISO/TS 10303-1034;
  • items within the scope of application module Time interval assignment, ISO/TS 10303-1365;
  • items within the scope of application module Work order, ISO/TS 10303-1043.
