TECHNICAL SPECIFICATION ISO/TS 10303-1246:2004(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1246:
Application module: Attribute classification

1 Scope

This part of ISO 10303 specifies the application module Attribute classification. The following are within the scope of this part of ISO 10303:

  • the association of a value, represented by a class, to an attribute on an entity.

    NOTE    the entities are defined by other modules that extend this module.

The following are outside the scope of this part of ISO 10303:

  • the existence of a class.
