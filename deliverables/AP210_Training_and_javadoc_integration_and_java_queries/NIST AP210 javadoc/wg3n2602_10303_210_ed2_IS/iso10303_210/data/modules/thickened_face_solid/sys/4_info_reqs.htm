4 Information requirements

This clause specifies the information requirements for the Thickened face solid application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Thickened_face_solid_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Thickened_face_solid_arm;
(*

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Solid_model_element   

A Solid_model_element is a type of Detailed_geometric_model_element. A Solid_model_element is a complete representation of the nominal shape of a product such that all points in the interior are connected. Any point can be classified as being inside, outside or on the boundary of a solid.

EXPRESS specification:

*)
ENTITY Solid_model_element
  SUBTYPE OF (Detailed_geometric_model_element);
END_ENTITY;
(*

4.2.2 Thickened_face_solid_model_element   

A Thickened_face_solid_model_element is a type of Solid_model_element that defines a solid by thickening a face on one or two sides.

EXPRESS specification:

*)
ENTITY Thickened_face_solid_model_element
  SUBTYPE OF (Solid_model_element);
  base_face : Advanced_face;
  base_thickness : length_measure;
  top_thickness : length_measure;
END_ENTITY;
(*

Attribute definitions:

base_face: the face to be thickened.

base_thickness: the depth of material to be added below the face to create a solid.

top_thickness: the depth of material to be added above the face to create a solid. The depth is measured in the direction of the surface normal to the face.

4.3 ARM subtype constraint definition

This subclause specifies the ARM subtype constraint for this module. The subtype constraint places a constraint on the possible super-type / subtype instantiations. The ARM subtype constraint and definition is specified below.

4.3.1 tbsa_detailed_geometric_model_element_subtypes   

The tbsa_detailed_geometric_model_element_subtypes constraint specifies a constraint that applies to instances of subtypes of Detailed_geometric_model_element.

EXPRESS specification:

*)
SUBTYPE_CONSTRAINT tbsa_detailed_geometric_model_element_subtypes FOR Detailed_geometric_model_element;
  ONEOF (Advanced_face,
         Axis_placement,
         Cartesian_point,
         Cartesian_transformation_2d,
         Cartesian_transformation_3d,
         Direction,
         Solid_model_element);
END_SUBTYPE_CONSTRAINT;
