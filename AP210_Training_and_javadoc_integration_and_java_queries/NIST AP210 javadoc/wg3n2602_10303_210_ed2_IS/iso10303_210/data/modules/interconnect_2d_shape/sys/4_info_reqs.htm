4 Information requirements

This clause specifies the information requirements for the Interconnect 2D shape application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Interconnect_2d_shape_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Interconnect_2d_shape_arm;
(*

4.1 Required AM ARMs

The following EXPRESS interface statements specify the elements imported from the ARMs of other application modules.

EXPRESS specification:

*)

4.2 ARM entity definition

This subclause specifies the ARM entity for this module. The ARM entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entity and definition is specified below.

4.2.1 Component_2d_embedded_location   

A Component_2d_embedded_location is a type of Component_2d_location. The Component_2d_embedded_location places components in a substrate under the constraint that is is required to maintain contact with Stratum_surface on both the top and bottom of the component.

NOTE    The Embedded_physical_component_terminal referenced by the Fabrication_joint referenced by reference_terminal should be a constituent of one of the surfaces referenced by precedent_surface or subsequent_surface.

EXPRESS specification:

*)
ENTITY Component_2d_embedded_location
  SUBTYPE OF (Component_2d_location);
  precedent_surface : Adjacent_stratum_surface_embedded_physical_component_surface_definition;
  subsequent_surface : Adjacent_stratum_surface_embedded_physical_component_surface_definition;
  reference_terminal : Fabrication_joint;
UNIQUE
  UR1: precedent_surface, reference_terminal, subsequent_surface;
WHERE
  WR1: precedent_surface <> subsequent_surface;
  WR2: SELF\Component_2d_location.substrate_location = FALSE;
  WR3: NOT EXISTS(SELF\Representation_relationship.description);
END_ENTITY;
(*

Attribute definitions:

precedent_surface: specifies the Adjacent_stratum_surface_embedded_physical_component_surface_definition that joins the Component_feature of the component being located to the associated stratum surface. The Adjacent_stratum_surface_embedded_physical_component_surface_definition in this role is identifying the stratum surface that is closer to the primary stratum in the substrate than is the embedded component itself.

subsequent_surface: specifies the Adjacent_stratum_surface_embedded_physical_component_surface_definition that joins the Component_feature of the component being located to the associated stratum surface. The Adjacent_stratum_surface_embedded_physical_component_surface_definition in this role is identifying the stratum surface that is further away from the primary stratum in the substrate than is the embedded component itself.

reference_terminal: specifies the Fabrication_joint that joins the Primary_reference_terminal of the component being located to the Land_join_terminal that implements the internal connection to the circuitry in the rest of the substrate.

Formal propositions:

UR1: The combination of precedent_surface, reference_terminal, and subsequent_surface shall be unique within a population of Component_2d_embedded_location.

WR1: The precedent_surface and subsequent_surface shall not be the same surface.

WR2: The substrate_location inherited from the Component_2d_location shall be "False".

WR3: The description shall not be populated.

4.3 ARM function definitions

This subclause specifies the ARM functions for this module. The ARM functions and definitions are specified below.

4.3.1 in_2d_assembly_context

The in_2d_assembly_context function returns TRUE only if specified Area_component has at least one Assembly_component_2d_shape_model referencing it as shape_characterized_component and if all such Assembly_component_2d_shape_models share context_of_items with at least one Physical_unit_planar_shape_model of assembly they are assembled into. Otherwise the function returns FALSE.

EXPRESS specification:

*)
FUNCTION in_2d_assembly_context (comp : Assembly_component) :LOGICAL; 
LOCAL
    shapes : SET OF Assembly_component_2d_shape_model := [];
    shapes_in_context : SET OF Physical_unit_planar_shape_model := [];
    assemblies : SET OF Product_view_definition := [];
    i, j : INTEGER := 0;
    in_context : LOGICAL := TRUE;
  END_LOCAL;
  
  REPEAT i := 1 TO SIZEOF(comp.occurrence_contexts);
    assemblies := assemblies + comp.occurrence_contexts[i].relating_view;
  END_REPEAT;

  shapes := bag_to_set(USEDIN(comp,'PHYSICAL_UNIT_2D_DESIGN_VIEW_ARM.ASSEMBLY_COMPONENT_2D_SHAPE_MODEL.SHAPE_CHARACTERIZED_COMPONENT'));
  REPEAT i := 1 TO SIZEOF(shapes);
    shapes_in_context := bag_to_set(USEDIN(shapes[i]\Geometric_model.context_of_items,'ELEMENTAL_GEOMETRIC_SHAPE_ARM.GEOMETRIC_MODEL.CONTEXT_OF_ITEMS'));
    in_context := FALSE;
    j := 1;
    REPEAT WHILE ((j <= SIZEOF(shapes_in_context)) AND (NOT(in_context)));
      in_context := (shapes_in_context[j].shape_characterized_definition IN assemblies);
      j := j + 1;
    END_REPEAT;
    
    IF NOT (in_context) THEN
      RETURN (FALSE);
    END_IF;
  END_REPEAT;
  
  RETURN (SIZEOF(shapes) > 0);
END_FUNCTION;
(*

Argument definitions:

comp: the specified Assembly_component.

4.3.2 no_2d_location_for_component

The no_2d_location_for_component function returns TRUE only if there is no Component_2d_location given for the specified component. Otherwise it returns FALSE.

EXPRESS specification:

*)
FUNCTION no_2d_location_for_component (comp : Assembly_component) :BOOLEAN; 
LOCAL
    i, j : INTEGER;
    cis : SET OF Contextual_item_shape;
    loc : SET OF Component_2d_location;
  END_LOCAL;
  
  REPEAT i := 1 TO SIZEOF(comp.occurrence_contexts);
    cis := bag_to_set(USEDIN(comp.occurrence_contexts[i],'SHAPE_PROPERTY_ASSIGNMENT_ARM.CONTEXTUAL_ITEM_SHAPE.DESCRIBED_ELEMENT'));
    REPEAT j := 1 TO SIZEOF(cis);
      loc := bag_to_set(USEDIN(cis[j],'CONTEXTUAL_SHAPE_POSITIONING_ARM.CONTEXTUAL_SHAPE_REPRESENTATION.CONTEXTUAL_SHAPE'));
      IF (SIZEOF(loc) > 0) THEN
        RETURN (FALSE);
      END_IF;
    END_REPEAT;
  END_REPEAT;

  RETURN (TRUE);
END_FUNCTION;
(*

Argument definitions:

comp: the specified Assembly_component.

4.4 ARM rule definitions

This subclause specifies the ARM rules for this module. The ARM rules and definitions are specified below.

4.4.1 component_in_assembly_2d_context

The component_in_assembly_2d_context constrains each Area_component to have at least one Assembly_component_2d_shape_model referencing it as shape_characterized_component and if all such Assembly_component_2d_shape_models share context_of_items with at least one Physical_unit_planar_shape_model of assembly they are assembled into.

EXPRESS specification:

*)
RULE component_in_assembly_2d_context FOR
(Area_component);
WHERE
  WR1: SIZEOF(QUERY( ac <* Area_component | NOT( (SIZEOF(USEDIN(ac,'PHYSICAL_UNIT_2D_DESIGN_VIEW_ARM.ASSEMBLY_COMPONENT_2D_SHAPE_MODEL.SHAPE_CHARACTERIZED_COMPONENT')) = 0) OR (in_2d_assembly_context(ac))) )) = 0;
END_RULE;
(*

Argument definitions:

Area_component : the set of all instances of Area_component.

Formal propositions:

WR1: Returns TRUE if each Area_component has at least one Assembly_component_2d_shape_model referencing it as shape_characterized_component and if all such Assembly_component_2d_shape_models share context_of_items with at least one Physical_unit_planar_shape_model of assembly they are assembled into.

4.4.2 no_component_assembly_2d_location_for_area_component

The no_component_assembly_2d_location_for_area_component rule constrains each Area_component to prohibit it from being located in 2D by a Component_2d_location.

NOTE    An Area_component is geometrically defined in place.

EXPRESS specification:

*)
RULE no_component_assembly_2d_location_for_area_component FOR
(Area_component);
WHERE
  WR1: SIZEOF(QUERY( ac <* Area_component | NOT( (SIZEOF(USEDIN(ac,'PHYSICAL_UNIT_2D_DESIGN_VIEW_ARM.ASSEMBLY_COMPONENT_2D_SHAPE_MODEL.SHAPE_CHARACTERIZED_COMPONENT')) = 0) OR (no_2d_location_for_component(ac))) )) = 0;
END_RULE;
(*

Argument definitions:

Area_component : the set of all instances of Area_component.

Formal propositions:

WR1: Each Area_component shall not be referenced by any Assembly_component_2d_shape_model as shape_characterized_component.



*)
END_SCHEMA;  -- Interconnect_2d_shape_arm
