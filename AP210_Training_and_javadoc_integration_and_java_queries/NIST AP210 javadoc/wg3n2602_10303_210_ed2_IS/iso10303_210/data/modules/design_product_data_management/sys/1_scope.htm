TECHNICAL SPECIFICATION ISO/TS 10303-1628:2009(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1628:
Application module: Design product data management

1 Scope

This part of ISO 10303 specifies the application module Design product data management. The following are within the scope of this part of ISO 10303:

  • product data management;
  • design data management;
  • items within the scope of application module Activity method assignment, ISO/TS 10303-1249;
  • items within the scope of application module Attribute classification, ISO/TS 10303-1246;
  • items within the scope of application module Classification with attributes, ISO/TS 10303-1111;
  • items within the scope of application module Generic material aspects, ISO/TS 10303-1681;
  • items within the scope of application module Information rights, ISO/TS 10303-1241;
  • items within the scope of application module Item definition structure, ISO/TS 10303-1345;
  • items within the scope of application module Pre defined product data management specializations, ISO/TS 10303-1760;
  • items within the scope of application module Product data management, ISO/TS 10303-1231;
  • items within the scope of application module Requirement assignment, ISO/TS 10303-1233;
  • items within the scope of application module Requirement view definition relationship, ISO/TS 10303-1142;
  • items within the scope of application module Specification control, ISO/TS 10303-1112;
  • items within the scope of application module Test select product, ISO/TS 10303-1757.

The following are outside the scope of this part of ISO 10303:

  • shape properties;
  • functional properties.
