4 Information requirements

This clause specifies the information requirements for the Swept solid application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Swept_solid_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Swept_solid_arm;
(*

4.1 Required AM ARM

EXPRESS specification:

*)
USE FROM Topologically_bounded_surface_arm;    --  ISO/TS 10303-1511

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Axis1_placement   

An Axis1_placement is a type of Detailed_geometric_model_element that defines the position of a single axis in 3 dimensional space.

EXPRESS specification:

*)
ENTITY Axis1_placement
  SUBTYPE OF (Detailed_geometric_model_element);
  axis : Direction;
  position : Cartesian_point;
END_ENTITY;
(*

Attribute definitions:

axis: the direction of the axis.

position: a reference point lying on the axis.

4.2.2 Extruded_solid   

An Extruded_solid is a type of Swept_solid formed by extruding the swept face along a straight line for a distance equal to the depth.

EXPRESS specification:

*)
ENTITY Extruded_solid
  SUBTYPE OF (Swept_solid);
  depth : length_measure;
  extruded_direction : Direction;
END_ENTITY;
(*

Attribute definitions:

depth: the distance the face is to be swept, this is also the length of all the new edges created by the extrusion operation.

extruded_direction: the direction in which the face is swept to create the solid; this direction shall not be perpendicular to the face normal.

4.2.3 Solid_of_revolution   

A Solid_of_revolution is a type of Swept_solid formed by revolving the swept planar face about an axis in the plane of the face. The axis shall not intersect the interior of the face.

EXPRESS specification:

*)
ENTITY Solid_of_revolution
  SUBTYPE OF (Swept_solid);
  angle : plane_angle_measure;
  axis : Axis1_placement;
END_ENTITY;
(*

Attribute definitions:

angle: the angle through which the face is revolved to create the solid. This angle is measured from the plane of the swept_face.

axis: the axis about which the face is revolved to generate the solid. The axis shall lie in the plane of the face.

4.2.4 Swept_solid   

A Swept_solid is a type of Detailed_geometric_model_element that defines a solid produced by sweeping a face. The face is based on a surface and is bounded by edges and vertices. The face is required to be an Advanced_face. A Swept_solid may be either an Extruded_solid or a Solid_of_revolution.

The following constraints apply:

  • The face is required to be fully bounded by edge loops;
  • At most one of the bounds of the face may define the outer bound;
  • The surface geometry of the face is required to be a plane.
  • The only types of sweeping operation permitted are linear extrusion or simple rotation about an axis.
  • The geometry of all bounding edges of the face shall be fully defined as edge curves;
  • The types of curve used to define the geometry of edges of the swept face shall be restricted to lines, conics, polylines, surface curves, or b spline curves;
  • All vertices used in the face definition shall be of type vertex point with geometry defined by a cartesian point;
  • The use of oriented paths in the definition of the edge loops of the advanced face is prohibited;
  • If a polyline is used as part of a face bound, it shall contain at least three points;

EXPRESS specification:

*)
ENTITY Swept_solid
  SUPERTYPE OF (ONEOF (Extruded_solid,
                       Solid_of_revolution))
  SUBTYPE OF (Detailed_geometric_model_element);
  swept_face : Advanced_face;
END_ENTITY;
(*

Attribute definitions:

swept_face: the face defining the area to be swept to create the solid. The swept_face may contain one or more holes.
