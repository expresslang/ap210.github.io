4 Information requirements

This clause specifies the information requirements for the Mechanical design geometric presentation application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Mechanical_design_geometric_presentation_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Mechanical_design_geometric_presentation_arm;
(*

4.1 Required AM ARMs

EXPRESS specification:

*)
USE FROM Camera_view_3d_arm;    --  ISO/TS 10303-1316

USE FROM Shape_appearance_layers_arm;    --  ISO/TS 10303-1009

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Mechanical_design_view_area   

A Mechanical_design_view_area is a type of View_area.

EXPRESS specification:

*)
ENTITY Mechanical_design_view_area
  SUBTYPE OF (View_area);
END_ENTITY;
(*

4.2.2 Mechanical_representation_styled_model   

A Mechanical_representation_styled_model is a type of Styled_model.

EXPRESS specification:

*)
ENTITY Mechanical_representation_styled_model
  SUBTYPE OF (Styled_model);
END_ENTITY;
