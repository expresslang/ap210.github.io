INTERNATIONAL STANDARD ISO 10303-503:2010(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 503:
Application interpreted construct: Geometrically bounded 2d wireframe

1 Scope

This part of ISO 10303 specifies the interpretation of the integrated resources to satisfy requirements for the representation of geometrically bounded 2d wireframe. The following are within the scope of this part of ISO 10303:
  • points defined in a two-dimensional coordinate space;
  • curves defined in a two-dimensional coordinate space that are trimmed by points or parameters;
  • curves defined in a two-dimensional coordinate space that are self-intersecting;
  • the representation of a single wireframe model or an assembly of wireframe models.

The following are outside the scope of this part of ISO 10303:

  • geometry defined in a three-dimensional coordinate space;
  • curves that are not trimmed and that are not self-intersecting.
