TECHNICAL SPECIFICATION ISO/TS 10303-1718:2009(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1718:
Application module: Layered interconnect simple template

1 Scope

This part of ISO 10303 specifies the application module Layered interconnect simple template. The following are within the scope of this part of ISO 10303:

  • Path based parametric curve including optional start, end and corner conditions;
  • Cross section parameters;
  • text template;
  • predefined special function templates;
  • template breakout pattern;
  • teardrop template;
  • trace template;
  • items within the scope of application module Fabrication technology, ISO/TS 10303-1670;
  • items within the scope of application module Fill area style, ISO/TS 10303-1672;
  • items within the scope of application module Layered 2d shape, ISO/TS 10303-1693;
  • items within the scope of application module Part template shape with parameters, ISO/TS 10303-1720;
  • items within the scope of application module Text representation, ISO/TS 10303-1750.

The following are outside the scope of this part of ISO 10303:

  • Variable thickness within a segment;
  • Variable width within a segment;
  • Multiple segments with different properties for each segment combined into one shape.
