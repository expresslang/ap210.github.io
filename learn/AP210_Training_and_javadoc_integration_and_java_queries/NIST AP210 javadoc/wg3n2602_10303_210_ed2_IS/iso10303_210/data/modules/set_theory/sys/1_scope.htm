TECHNICAL SPECIFICATION ISO/TS 10303-1210:2004(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1210:
Application module: Set theory

1 Scope

This part of ISO 10303 specifies the application module Set theory. The following are within the scope of this part of ISO 10303:

  • complement;
  • intersection;
  • power set;
  • proper subset;
  • same membership (that is equality);
  • subset;
  • union.

The following are outside the scope of this part of ISO 10303:

  • the definition of a set or class;

    NOTE    Definition of a set or class is within the scope of the Class application module.

  • classification and the specification of complete set membership;
  • mappings between sets or classes;
  • relationships that are not set theory relationships.
