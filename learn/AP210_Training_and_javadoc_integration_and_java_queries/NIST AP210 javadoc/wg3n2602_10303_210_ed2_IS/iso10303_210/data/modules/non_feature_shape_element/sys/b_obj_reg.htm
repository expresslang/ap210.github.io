Annex B
(normative)

Information object registration

B.1 Document identification

To provide for unambiguous identification of an information object in an open system, the object identifier

{ iso standard 10303 part(1706) version(2) }

is assigned to this part of ISO 10303. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.

B.2 Schema identification

B.2.1 Non_feature_shape_element_arm schema identification

To provide for unambiguous identification of the schema specifications given in this application module in an open information system, the object identifiers are assigned as follows:

{ iso standard 10303 part(1706) version(2) schema(1) non-feature-shape-element-arm(1) }

is assigned to the Non_feature_shape_element_arm schema. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.

B.2.2 Non_feature_shape_element_mim schema identification

To provide for unambiguous identification of the schema specifications given in this application module in an open information system, the object identifiers are assigned as follows:

{ iso standard 10303 part(1706) version(2) schema(1) non-feature-shape-element-mim(2) }

is assigned to the Non_feature_shape_element_mim schema. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.
