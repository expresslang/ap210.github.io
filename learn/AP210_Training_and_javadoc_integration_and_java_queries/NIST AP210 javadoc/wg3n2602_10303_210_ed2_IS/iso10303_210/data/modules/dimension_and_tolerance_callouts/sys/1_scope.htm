TECHNICAL SPECIFICATION ISO/TS 10303-1362:2008(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1362:
Application module: Dimension and tolerance callouts

1 Scope

This part of ISO 10303 specifies the application module Dimension and tolerance callouts. The following are within the scope of this part of ISO 10303:

  • the representation of annotation elements in an explicit form used in technical drawings and annotated geometric models;
  • the representation of annotation elements for dimension and dimensional tolerance callouts;
  • the representation of annotation elements for geometric tolerance and datum feature callouts;
  • the representation of annotation elements for surface condition callout;
  • items within the scope of application module Associative draughting elements, ISO/TS 10303-1311;
  • items within the scope of application module Dimension tolerance, ISO/TS 10303-1050;
  • items within the scope of application module Draughting annotation, ISO/TS 10303-1206;
  • items within the scope of application module Draughting element specialisations, ISO/TS 10303-1312.

The following are outside the scope of this part of ISO 10303:

  • associativity of annotation elements other than text.
