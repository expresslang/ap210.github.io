TECHNICAL SPECIFICATION ISO/TS 10303-1026:2008(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1026:
Application module: Assembly structure

1 Scope

This part of ISO 10303 specifies the application module Assembly structure. The following are within the scope of this part of ISO 10303:

  • specification of the use of a product version as a component of an assembly;
  • specification of a parent-child assembly relationship;
  • identification of a component of an assembly with respect to an upper level in the assembly structure;
  • items within the scope of application module Product view definition, ISO/TS 10303-1019;
  • items within the scope of application module Value with unit, ISO/TS 10303-1054.

The following are outside the scope of this part of ISO 10303:

  • characterization of the position of a component in an assembly;

    NOTE    This aspect is specified in the Contextual shape positioning module.

  • characterization of a component quantity with a qualified quantity or a quantity range.
