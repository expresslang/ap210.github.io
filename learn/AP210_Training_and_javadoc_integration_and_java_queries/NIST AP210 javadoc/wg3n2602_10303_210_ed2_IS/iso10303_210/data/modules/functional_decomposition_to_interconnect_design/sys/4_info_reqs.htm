4 Information requirements

This clause specifies the information requirements for the Functional decomposition to interconnect design application module. The information requirements are specified as the Application Reference Model (ARM) of this application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met by using common resources and constructs defined or imported in the MIM schema of this application module.

The following EXPRESS specification begins the Functional_decomposition_to_interconnect_design_arm schema and identifies the necessary external references.

EXPRESS specification:

*)
SCHEMA Functional_decomposition_to_interconnect_design_arm;
(*

4.2 ARM entity definitions

This subclause specifies the ARM entities for this module. Each ARM application entity is an atomic element that embodies a unique application concept and contains attributes specifying the data elements of the entity. The ARM entities and definitions are specified below.

4.2.1 Design_functional_unit_allocation_to_fabrication_joint   

A Design_functional_unit_allocation_to_fabrication_joint is an assignment of a Functional_unit in a Design_composition_path to its implementation in a Fabrication_joint.

EXPRESS specification:

*)
ENTITY Design_functional_unit_allocation_to_fabrication_joint;
  functional_design_definition_path : Design_composition_path;
  implementation : Fabrication_joint;
UNIQUE
  UR1: functional_design_definition_path;
END_ENTITY;
(*

Attribute definitions:

functional_design_definition_path: specifies the Design_composition_path for the Design_functional_unit_allocation_to_fabrication_joint.

implementation: specifies the Fabrication_joint for the Design_functional_unit_allocation_to_fabrication_joint.

Formal propositions:

UR1: The functional_design_definition_path shall be unique within a population of Design_functional_unit_allocation_to_fabrication_joint.

4.2.2 Transmission_line_functional_unit_terminal_allocation   

A Transmission_line_functional_unit_terminal_allocation is the relationship between the occurrence of a Functional_unit_terminal of a transmission line and the physical implementation of that occurrence as a Printed_component_join_terminal.

EXPRESS specification:

*)
ENTITY Transmission_line_functional_unit_terminal_allocation;
  implementation : Printed_component_join_terminal;
  transmission_line_functional_unit_terminal : Functional_unit_terminal;
  functional_unit_design_path : Transmission_line_functional_unit_to_routed_transmission_line_allocation;
UNIQUE
  UR1: implementation, transmission_line_functional_unit_terminal;
END_ENTITY;
(*

Attribute definitions:

implementation: specifies a role of the Printed_component_join_terminal for the Transmission_line_functional_unit_terminal_allocation.

transmission_line_functional_unit_terminal: specifies a role of the Functional_unit_terminal for this Transmission_line_functional_unit_terminal_allocation.

functional_unit_design_path: specifies the Transmission_line_functional_unit_to_routed_transmission_line_allocation path from the Functional_unit_terminal to the Printed_component_join_terminal for the Transmission_line_functional_unit_terminal_allocation.

Formal propositions:

UR1: The combination of the implementation and transmission_line_functional_unit_terminal shall be unique within a population of Transmission_line_functional_unit_terminal_allocation.

4.2.3 Transmission_line_functional_unit_to_routed_transmission_line_allocation   

A Transmission_line_functional_unit_to_routed_transmission_line_allocation is a type of Design_functional_unit_allocation_to_assembly_component. A Transmission_line_functional_unit_to_routed_transmission_line_allocation is the allocation of the functional requirements specified by a member of Functional_unit_definition in a hierarchical definition to the Conductive_interconnect_element.

EXPRESS specification:

*)
ENTITY Transmission_line_functional_unit_to_routed_transmission_line_allocation
  SUBTYPE OF (Design_functional_unit_allocation_to_assembly_component);
  SELF\Design_functional_unit_allocation_to_assembly_component.implementation : Routed_transmission_line;
END_ENTITY;
(*

Attribute definitions:

implementation: specifies a role of the Routed_transmission_line for the Transmission_line_functional_unit_to_routed_transmission_line_allocation.



*)
