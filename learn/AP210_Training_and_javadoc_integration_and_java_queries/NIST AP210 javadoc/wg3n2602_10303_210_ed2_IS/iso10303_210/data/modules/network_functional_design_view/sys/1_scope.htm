TECHNICAL SPECIFICATION ISO/TS 10303-1704:2009(E)

Industrial automation systems and integration —
Product data representation and exchange —
Part 1704:
Application module: Network functional design view

1 Scope

This part of ISO 10303 specifies the application module Network functional design view. The following are within the scope of this part of ISO 10303:

  • identification of a functional unit;
  • identification and representation of a network of functional units;
  • identification and representation of connections between functional units in the network;
  • identification of the equivalent functional unit definitions;
  • identification of the binding between a functional node in a network and the terminal declaration on the interface;
  • items within the scope of application module Assembly structure, ISO/TS 10303-1026;
  • items within the scope of application module Functional usage view, ISO/TS 10303-1705;
  • items within the scope of application module Generic product occurrence, ISO/TS 10303-1762;
  • items within the scope of application module Requirement assignment, ISO/WD 10303-1233.

The following are outside the scope of this part of ISO 10303:

  • usage of a function;
  • schematic or pictorial representation of a network;
  • packaging applications.
